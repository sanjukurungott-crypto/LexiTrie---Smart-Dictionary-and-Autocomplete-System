# LexiTrie – Smart Dictionary & Autocomplete System

A modern, responsive web application designed as a frontend sandbox and explorer for a C++ **Trie (Prefix Tree)** data structure. The visual architecture is inspired by premium productivity suites like Google Workspace and Notion.

## Features

- **Dashboard:** Interactive statistics (word counts, search frequency counts, active sandbox status) and recent query logs.
- **Search Dictionary:** Autocomplete prefix suggestions (matching typed roots instantly) and typos spell-correction checks.
- **Dictionary Manager:** Modern operations panel supporting insertion, deletion, and real-time filtering of dictionary contents.
- **Search History:** Historical timeline of query resolutions and audit logging.
- **Statistics Visualizations:** Real-time data plots powered by **Chart.js** displaying search distributions and resolution success rates.
- **Trie visualizer:** Live interactive visual tree explaining how prefix links and terminal endpoints are connected.
- **C++ Backend Configuration:** Simple modal config for routing web endpoints to a local C++ server.
